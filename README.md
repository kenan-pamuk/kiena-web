# Kiena Next.js + Tailwind Landing Page

## Çalıştırma

```bash
npm install
npm run dev
```

Tarayıcıda `http://localhost:3000` adresini açın.

## Not
Bu proje, gönderdiğiniz ekran görüntüsünü referans alarak hazırlanmış Next.js + Tailwind + TypeScript landing page taslağıdır. `public/assets/reference.png` dosyası karşılaştırma amaçlı eklenmiştir.
